4.List any 3 ways which help us in improving the accessibility in html.

solution
Improving accessibility in HTML involves various techniques and best practices to ensure that web content is usable by a wide range of people, including those with disabilities. Here are three important ways to enhance accessibility in HTML:

1.Semantic HTML Elements: Use semantic HTML elements to provide meaningful structure to your web content. This includes elements like <header>, <nav>, <main>, <article>, <section>, and <footer>. Semantic elements help screen readers and other assistive technologies understand the document's structure and content, making it easier for users to navigate and comprehend.

2.Alt Text for Images: Always include descriptive alternative text (alt text) for images using the alt attribute in the <img> element. Alt text provides a textual description of the image's content, making it accessible to individuals who cannot see the image due to visual impairments or other reasons. Ensure that alt text conveys the essential information and context of the image.

3.`Proper Form Markup: When creating forms in HTML, use proper form elements and attributes to enhance accessibility. This includes labeling form controls with <label> elements using the for attribute, associating labels with form elements through the id attribute, and providing additional instructions and context using elements like <fieldset> and <legend>. Using appropriate input types and attributes for fields (e.g., type="text", type="email", type="tel", type="checkbox", etc.) also helps improve accessibility and user experience.

By incorporating these practices into your HTML code, you can significantly improve the accessibility of your web content, making it more inclusive and user-friendly for individuals with disabilities.
